# Mission Control

Mission Control is a dark operational dashboard for projects, agents, delivery visibility, and future orchestration.

This repository is the first product implementation for the broader team operating system defined in the OpenClaw workspace docs.

## Current direction

Version 1 focuses on clarity and structure.

The first release is intended to provide:
- an overview of current work
- active project visibility
- team role visibility
- sprint and blocker awareness
- quick links into the surrounding operating system

## Stack

- Next.js
- TypeScript
- Tailwind CSS

## Local development

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```
